## Parte 1
Implemente las siguientes funciones en Haskell y pruebe su ejecución:

1. F(x) = 2x + 3 
2. G(x, y) = 2x + 3y + 5 
3. H(z) = z^2 + 12 
4. I(x, y) = (12x + 15y) (x + y + z)

Tome en cuenta que el operador para el cálculo de potencia es "^".
